<center>
Описание функциональности
</center>
Программа вычисляет среднее геометрическое двух действительных чисел,требуется проверка функциональности для отрицательных чисел и типов данных помимо int и float. Программа учитывает, ситуации, в которых пользователь вводит аргументы, несоответствующие типам аргументов в функции, а также некорректные значения переменных.
<center>
Области проверки
</center>
Типы и значения вводимых переменных.
<center>
Стратегия
</center>
Функциональное тестирование: Проверка того, что вывод функции соответствует ожидаемому результату.<br>
Тестирование безопасности : Проверка на совместимость типов и входных значений.
